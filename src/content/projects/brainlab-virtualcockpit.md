As an R&D Software Engineer at Brainlab AG, I developed an end-to-end augmented reality system that mirrors multiple operating room data sources into a single Virtual Cockpit. The application combines live fluoroscopy, preoperative plans, and 3D anatomy reconstructions in a configurable spatial layout tailored to the surgeon and the clinical setup.


## Highlights
- Built the data pipeline to retrieve live intraoperative imaging, preoperative plans, and patient-specific 3D reconstructions.
- Developed a Magic Leap 2 interface with movable virtual screens and a customizable layout for different operating room settings.
- Collaborated with clinicians from Heidelberg University Hospital during clinical validation sessions.
